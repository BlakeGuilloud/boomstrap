btAddClassOnLoad
directive in module boomstrap

Description

The btAddClassOnLoad directive adds a class to an img tag when the image fires the load event. This is useful for css transitions.

Usage

as attribute
<img bt-add-class-on-load="{string}">
   ...
</img>

Parameters

ParamTypeDetails
btAddClassOnLoadstring

The string representation of the class to add to the image
