btNgModelOnBlur
directive in module boomstrap

Description

The btNgModelOnBlur attribute directive when used with an input will only update the ngModel when the user has left focus of the input or pressed the enter key. This directive will be made redundant with the ngModelOptions directive in Angular 1.3

Dependencies

  • ngModel

Usage

as attribute
<ANY bt-ng-model-on-blur>
   ...
</ANY>
