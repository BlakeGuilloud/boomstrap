phoneNumber

Description

The 'phoneNumberfilter' filter will format or unformat a phone number. {{string|phoneNumber:'add'}} to format the number, {{string|phoneNumber:'remove'}}.

Usage

In HTML Template Binding

{{ phoneNumber_expression | phoneNumber }}

In JavaScript

$filter('phoneNumber')()

Example

$scope.string1 = '1234567890';
$scope.string2 = '123-456-7890';

{{string1|phoneNumber:'add'}} // outputs (123) 456-7890
{{string2|phoneNumber:'remove'}} // outputs 1234567890
