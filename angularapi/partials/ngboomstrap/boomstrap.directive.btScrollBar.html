btScrollBar
directive in module boomstrap

Description

The btScrollBar directive adds a simulated scroll-bar to any element. It wraps the jQuery baron library.

Usage

as attribute
<ANY bt-scroll-bar
     total-items="{Number}"
     current-page="{Number}"
     items-per-page="{Number}">
   ...
</ANY>

Parameters

ParamTypeDetails
totalItemsNumber

The number of items to paginate through.

currentPageNumber

The number of the current page we are on.

itemsPerPageNumber

How many items are allowed on each page.
