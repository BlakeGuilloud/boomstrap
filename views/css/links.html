Links

Default links

Simple use of anchor tags.

Hello there! Visit the BoomTown website to learn more about our company!

<p>Hello there! Visit the <a href="http://boomtownroi.com/" target="_blank">BoomTown website</a> to learn more about our company!</p>

Clickable links

Sometimes we want anchor tag styles without the anchor tag markup. Use .clickable instead.

Hello there! Visit the BoomTown website to learn more about our company!

<p>Hello there! Visit the <span class="clickable">BoomTown website</span> to learn more about our company!</p>

Use .clickable .clickable--muted for a muted version that inherits .text-muted contextual color styles.

Hello there! Visit the BoomTown website to learn more about our company!

<p>Hello there! Visit the <span class="clickable clickable--muted">BoomTown website</span> to learn more about our company!</p>
