<h3>Example heading <span class="label label-default">New</span></h3>

Available variations

Add any of the below mentioned modifier classes to change the appearance of a label.

Default Primary Success Info Warning Danger 
<span class="label label-default">Default</span>
<span class="label label-primary">Primary</span>
<span class="label label-success">Success</span>
<span class="label label-info">Info</span>
<span class="label label-warning">Warning</span>
<span class="label label-danger">Danger</span>

Lead categories

Lead category labels extend the .label class.

Default

archive closed hot new nurture pending qualify trash watch 

<span class="cat cat-archive">archive</span>
<span class="cat cat-closed">closed</span>
<span class="cat cat-hot">hot</span>
<span class="cat cat-new">new</span>
<span class="cat cat-nurture">nurture</span>
<span class="cat cat-pending">pending</span>
<span class="cat cat-qualify">qualify</span>
<span class="cat cat-trash">trash</span>
<span class="cat cat-watch">watch</span>

Equal width

For equal width lead category labels, use the .cat-eq class.

archive closed hot new nurture pending qualify trash watch 

<span class="cat cat-eq cat-archive">archive</span>
<span class="cat cat-eq cat-closed">closed</span>
<span class="cat cat-eq cat-hot">hot</span>
<span class="cat cat-eq cat-new">new</span>
<span class="cat cat-eq cat-nurture">nurture</span>
<span class="cat cat-eq cat-pending">pending</span>
<span class="cat cat-eq cat-qualify">qualify</span>
<span class="cat cat-eq cat-trash">trash</span>
<span class="cat cat-eq cat-watch">watch</span>

Equal width abbreviated

For equal width lead category labels that are abbreviated, use the .cat-eq-abbr class.

arch close hot new nurt pend qual trash watch

<span class="cat cat-eq-abbr cat-qualify">qual</span>
<span class="cat cat-eq-abbr cat-trash">trash</span>
<span class="cat cat-eq-abbr cat-watch">watch</span>
