Deprecated Font Icons

We have switched to an SVG icon system. Font icons are still available for use but development has stopped and they will eventually be removed from the project - view font icons.

We have a serious set of SVG icons. We've implemented our icons with SVG 'use' pointing to an external source (see CSS Tricks and svg4everybody).
