Log call

Modular component for logging a call. Uses BoomQueries to apply size modifier classes.

BoomQuery Values

Breakpoint Size Modifier Class
>= 420px log-call--sm

Log call combined with set to-do component

This is the log call and set to-do component combined in a small modal. Simply add the necessary set to-do classes .set-to-do .set-to-do--modal .set-to-do--modal-sm to the existing log call component.

Interaction note (log call)

When a log call status button is clicked it will trigger the reveal of the set to-do componenet using the collapse plug-in.

Interaction note (set to-do)

When a set to-do date button is clicked, it should receive the .active class, change the displayed date accordingly, and the .hide class should be removed from .set-to-do__cancel.
Then, when the cancel to-do button is clicked, it should receive the .hide class and reset the displayed date accordingly.
