Log call in small modal

This is the log call component wrapped in a small modal. Add the .log-call--modal and .log-call--modal-sm modifier classes to the existing.log-call class.

Log call in modals

When using the log call component in a modal, add the .log-call--modal modifier class to the existing .log-call class. Currently there are no modifier classes needed for use within default and large modals.
