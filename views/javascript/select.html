Select bootstrap-select.js

Custom select using button dropdown. See http://silviomoreto.github.io/bootstrap-select/3/ for options and documentation.

<select class="selectpicker">
  <option>Mustard</option>
  <option>Ketchup</option>
  <option>Relish</option>
</select>

Call selectpicker manually:

$('.selectpicker').selectpicker()
