Element Query boomqueries.js

BoomQueries is a plugin used to perform element queries. See https://github.com/BoomTownROI/boomqueries for options and documentation.

About Element Queries

Element queries provide us with granular control over how the contents of an individual component will respond relative to its own size rather than the viewport size via media queries. To learn more about element queries, visit http://responsiveimagescg.github.io/eq-usecases/.
