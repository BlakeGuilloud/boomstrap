Element Queries boomqueries.js

BoomQueries is a plugin used to perform element queries. See https://github.com/BoomTownROI/boomqueries for options and documentation.

About Element Queries

Element queries provide us with granular control over how the contents of an individual component will respond relative to its own size rather than the viewport size via media queries. To learn more about element queries, visit http://responsiveimagescg.github.io/eq-usecases/.

Basic example

In the following example, we register the component (.pl-component) with BoomQueries via JavaScript:

boomQueries.add(".pl-component", [[420, "pl-component--sm"],[600, "pl-component--md"]]);

Once registered, the component is set up to receive size modifier classes when its width matches the defined breakpoints. In this example, when the component's width reaches 420px it will receive the .pl-component--sm class. When the component's width reaches 600px it will receive the .pl-component--md class.

For demonstration purposes, we've provided two visual cues that indicate when a modifier class is added or removed:
1) text inside of the component reveals modifier classes
2) component's background changes color
