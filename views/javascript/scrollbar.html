Scrollbar (perfect-scrollbar)

The image below sits in a <div> tag with class perfect-scrollbar. If the inner content is wider or taller than the .perfect-scrollbar container, scrollbars will be triggered.

Usage

Create a perfect-scrollbar container with a class to hook up some CSS.

HTML

<div class="perfect-scrollbar awesome-clizzass">
    <p>So Snoopy!</p>
</div>

CSS

/* set a width and height in your CSS */

.awesome-clizzass {
    width: 300px;
    height: 816px;
}

JavaScript

/* init */

(function($) {
    $(document).ready(function() {
        $('.perfect-scrollbar').perfectScrollbar({ wheelSpeed: 0.5 });
    });
})(jQuery);

/* for more options, see the perfect-scrollbar docs */

If your scrollbar container changes size (e.g. after an AJAX call), call $('.perfect-scrollbar').perfectScrollbar('update'); to reinitialize the container.
