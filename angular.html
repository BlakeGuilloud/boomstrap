UI-Bootstrap Components

Boomstrap and UI-Bootstrap

Our basic components are built atop the AngularUI team's UI-Bootstrap project but styled like the rest of the components in Boomstrap. The UI-Bootstrap components employed by Boomstrap are listed below. For background and documentation directly related to UI-Bootstrap, head over to the UI-Bootstrap page on GitHub.

Accordion, Alert, Buttons, Carousel, Collapse, Datepicker, Dropdown, Modal, Pagination, Popover, Tabs, Timepicker, Tooltip

Chosen (Chosen, angular-chosen)

Chosen allows for searchable dropdowns.

Provide a chosen attribute with an ng-options attribute to a select tag to search for elements.

<select chosen
  ng-model="superhero"
  ng-options="s for s in superheroes">
</select>
<option value=""></option>
<label>You have chosen {{ superhero }}!</label>
            
(function(PatternLibrary) {
  'use strict';
  PatternLibrary.controller('ChosenDemoCtrl', ['$scope', function($scope) {
    $scope.superheroes = [
      'Batman',
      'Superman',
      'Green Arrow',
      'Wonder Woman'
    ];
  }]);
})(angular.module('PatternLibrary'));
